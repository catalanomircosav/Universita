- $y = \log_a x \iff x = ay$
- $log_a 1 = 0$
- $loga a = 1$
- $\log_a (xy) = \log_a x + \log_a y$
- $\log_a (\frac{x}{y}) = \log_a x - \log_a y$
- $\log_a (xb) = b\cdot\log_a x$
- $\log_b x = \frac{\log_a x}{\log_a b}$ 
- $\log_x \quad\text{is typically}\quad \log_{10} x$
- $\ln x \quad\text{is typically}\quad log_e x$ $(e = 2.7182)$
